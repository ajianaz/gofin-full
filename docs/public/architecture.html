Gofin Architecture

Self-hosted personal finance tracker — Go API + SvelteKit + PostgreSQL + Redis

Frontend

  • • SvelteKit 5 with Svelte 5 runes
  • • 43 pages, file-based routing
  • • Tailwind CSS 4 + shadcn-svelte
  • • Chart.js analytics
  • • i18n: Indonesian + English
  • • Dark mode with system detection

Backend

  • • Go 1.25, Fiber v2 framework
  • • 135+ REST endpoints, 29 groups
  • • Layered: Handler → Service → Repo
  • • shopspring/decimal for money
  • • UUID v7 for all IDs
  • • Parameterized SQL (no injection)

Security

  • • JWT access + refresh tokens
  • • 21 group + 3 wallet RBAC levels
  • • OAuth2 (Google, GitHub)
  • • Password policy (bcrypt + strength)
  • • Rate limiting (Redis sliding window)
  • • HSTS, CORS restricted
